# Архив

> **Status:** current  
> **Last verified:** YYYY-MM-DD  
> **Owner:** @Pixasso

Устаревшая документация, исторические заметки и выполненные планы.

## Назначение

Эта папка содержит:
- **Устаревшие документы** — неактуальные, но потенциально полезные для справки
- **Заметки о правках** — исторические исправления багов и инциденты
- **Выполненные планы** — реализованные функции, перемещённые из `plans/`
- **Устаревшие руководства** — старые процессы, которые были заменены

## Соглашение об именовании

При архивировании документа:
1. Сохраняйте оригинальное имя файла
2. Добавьте суффикс `_ARCHIVED` или дату при необходимости
3. Добавьте заметку вверху, объясняющую причину архивирования

## Шаблон

При перемещении документа в архив добавьте этот заголовок:

```markdown
> **ARCHIVED:** YYYY-MM-DD  
> **Reason:** [Почему это больше не актуально]  
> **Replaced by:** [Ссылка на новый документ, если применимо]

---

[Оригинальное содержимое ниже]
```
